# 🗒️ Client Meeting — One Page (print this)

**Run order:** demo while you talk → then ask. Look like it's already built (it is).

## TELL them
1. **Reframe:** "A chatbot waits to be asked — farmers don't know what to ask. Ours tells them what to do and when, *before* they ask."
2. **Already live on WhatsApp:** no app, no reading — just WhatsApp + voice.
3. **3 wow moments (say during demo):** understood a *voice note* · gave *exact dates* · *changed the plan* for weather.
4. **Trusted, not guessing:** brain = verified university/KVK agronomy data + date math, not a hallucinating AI.
5. **Killer feature works:** proactive daily reminders before every task.

## ASK them
**Scope (pin first)**
1. Which crops + which district/region to prioritize?
2. Individual farmers or cooperatives/KVKs?

**Data & trust (protect yourself)**
3. Can you provide an **agronomist** to validate dates? (No go-live without it.)
4. Any existing schedules / soil data / university partnerships?
5. Adjust advice for soil type, organic vs chemical, irrigated vs rain-fed?

**Features (v1 vs later)**
6. First add-on: disease photo detection / market prices / weather / govt schemes?
7. Exact languages & dialects?
8. Must it work on basic phones / offline, or is WhatsApp+internet fine?

**Business**
9. Success metric — # farmers, yield, engagement?
10. Pilot or production? Budget + timeline?
11. Official WhatsApp Business API or Twilio sandbox for pilot?

## The one ask that matters most
**Agronomist + crop list + region.** Everything else I can wire up; only they can give region-correct, expert-validated dates.
